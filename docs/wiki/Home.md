## 📖 Core Concepts

### The Diátaxis Quadrants

Our documentation is organized around **four types of content**, each serving a different need:

```
                Learning ←──────→ Working
                    │               │
       TUTORIALS    │   HOW-TO      │
       (lessons)    │   (recipes)   │
                    │               │
    ────────────────┼───────────────┼───
                    │               │
     EXPLANATION    │   REFERENCE   │
     (discussion)   │   (facts)     │
                    │               │
                Theory ←──────→ Practice
```

### Why This Matters

- ✅ **Find what you need faster** - Clear separation of content types
- ✅ **No more confusion** - Tutorials don't try to be reference guides
- ✅ **Better learning** - Learn when you want, look up when you need
- ✅ **Deeper understanding** - Context separated from instructions
