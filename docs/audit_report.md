## 4. Potential Issues

### 4.1 Path Traversal in code_scan_handler
The path parameter in CodeScanPayload is not validated. A malicious actor could scan arbitrary paths outside the workspace.

**Recommendation:** Add path validation to ensure scanned paths are within allowed directories.

### 4.2 No Rate Limiting
All endpoints lack rate limiting. For client deployment, consider adding basic rate limiting.

### 4.3 No Authentication on HTTP Server
The HTTP server currently has no authentication. For production deployment, add token-based auth similar to Cortex.

### 4.4 code_find_handler and code_context_handler return 0 results
Despite code scan completing successfully (141 files, 2665 symbols indexed), queries return 0 results.
This suggests an issue with the code_query engine or index format mismatch.
